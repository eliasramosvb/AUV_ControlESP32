# AUV Control ESP32

This project implements a control system for a Mini AUV using an ESP32.

## Features
- Control of 3 DC motors
- Serial communication protocol
- Battery voltage monitoring
- Modular code structure

## Files
- AUV_ControlESP32.ino → main program
- Motors → motor control logic
- Battery → battery monitoring
- SerialProtocol → communication
